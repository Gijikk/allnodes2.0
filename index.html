Full Nodes

Full nodes are computers that store a separate copy of the blockchain on a hard drive, which is regularly updated using special software. These nodes guarantee the viability of the cryptocurrency network by participating in transaction processing and data relaying in the ecosystem. The owner of a full node does not receive dividends, but if you have invested in this cryptocurrency and you have the opportunity to run and maintain a node of this type, it is highly recommended to do so. A full node enforces the rules and protects the blockchain from hackers and scammers. In addition, full node is a reliable wallet for storing crypto assets.

Mining Nodes

To start mining cryptocurrency, you need a wallet. But not every miner runs a full node, which you need to install if you decide to organize your own pool, or to mine crypto in solo, using a network wallet ID instead of a pool address. Otherwise, the wallet can be of any type, albeit paper. Miners participate in transaction processing and block formation, but have the right to write a new block to the blockchain without the approval of other full nodes. Only after a consensus is reached, the block is recognized as valid and the miners receive money for it.

Masternodes

Master nodes together with miners receive part of the reward for the block, so it is profitable to keep such a package, unless, of course, this crypto project goes bankrupt. A master node is not even required on your own PC. This can be a remote server node that you rent from one provider or another. The main condition is a sufficient amount for freezing. You need either a local wallet with a full copy or a hardware safe connected to your computer. An example of a fall with the masternodes of this DASH cryptocurrency.

SPV Nodes

Light clients Simplified Payment Verification (SPV) are used to work with the network through a remote node. This is done so that applications can work with cryptocurrency without downloading the ledger to their gadget. Thanks to the launch of convenient nodes, multi-currency wallets, smartphone and browser applications have appeared. Lightweight SPV nodes use communication endpoints and are not considered for transaction processing and network security. Such clients are less reliable, but they are fast, compact and do not consume a lot of computing resources.
